# GlowTrap Studio Website

This is the GlowTrap Studio website.

## Website structure

- index.html
- perfume.html
- products.html
- about.html
- contact.html
- css/style.css
- js/main.js
- images/

## Notes

GlowTrap Studio is a luxury multi-product brand website.

- Home page = main brand page
- Perfume page = fragrance only
- Other Products page = beauty, body, accessories, and future drops
- About page = brand identity
- Contact page = inquiry / payment later

## Design style

- Obsidian black
- Deep plum smoke
- Champagne gold
- Soft ivory

## Important

GitHub Pages needs `index.html` at the top level to load the website correctly.
