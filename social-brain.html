SETUP GUIDE

Live in under 30 minutes.

Install n8n (self-hosted or cloud)
The fastest way is n8n Cloud — create a free account at app.n8n.cloud. Or self-host with Docker:
docker run -it --rm \ -p 5678:5678 \ -v n8n_data:/home/node/.n8n \ n8nio/n8n
Set up your Notion workspace
Create a new Notion page called Social Brain HQ. Inside it, create all 7 databases listed above. Use the database schema JSON as a reference for property types. Then create a Notion integration at notion.so/my-integrations, copy the API key, and share each database with the integration.
Configure environment variables
Download the .env.example file. Fill in your API keys for every service you want to use. In n8n, go to Settings → Environment Variables and add each one. At minimum you need: ANTHROPIC_API_KEY, RAPIDAPI_KEY, NOTION_API_KEY, and all 7 NOTION_*_DB IDs.
Import the 9 workflows into n8n
Download all workflow JSON files above. In n8n, go to Workflows → Import from File and upload each JSON one by one. They import in seconds. You'll see all 9 workflows in your dashboard.
Add your API credentials in n8n
In n8n, go to Credentials and add a Notion API credential using your integration token. The HTTP Request nodes use environment variables directly — no additional credential setup needed for those. Activate workflows 01, 02, 03, 08, and 09 (the scheduled ones) right away.
Set the creators you want to track
In your n8n environment variables, set TRACK_CREATORS to a comma-separated list of TikTok usernames (no @ symbol). Example: khaby.lame,mrbeast,yourcompetitor. The monitor workflow will analyze them daily and build intelligence.
Use the AI Content Brain
Send a POST request to your n8n webhook URL:
curl -X POST https://your-n8n.com/webhook/content-brain \ -H "Content-Type: application/json" \ -d '{ "topic": "I want to make content about morning routines for entrepreneurs", "platform": "tiktok", "style": "educational+entertaining" }'
Claude will pull in live trends, creator hooks, and your performance data, then return 5 viral ideas with full scripts and captions. The best one lands in your Notion pipeline automatically.
Approve content and let it run
Open your Notion Content Pipeline board. Review the AI-generated ideas. Change any idea's Status from Idea to Approved — this triggers Video Production automatically. Once the video is ready, the Smart Scheduler picks the best posting time, and the Cross-Platform Publisher fires at that time. You just watch the numbers.
