THE FIVE PHASES

From idea to production.

01
📋
Specification
Clarify objectives, constraints, and acceptance criteria. Define what success looks like before writing a single line of code.
  • Define functional requirements
  • Identify constraints and edge cases
  • Write acceptance criteria
  • No hardcoded env vars or secrets
02
🧠
Pseudocode
Create high-level logic flows and algorithmic planning with TDD anchors. Think before you code.
  • High-level algorithmic design
  • Data flow mapping
  • TDD anchor points (test-first)
  • Edge case identification
03
🏗️
Architecture
Design extensible system diagrams, service boundaries, and API contracts. Build for scale from day one.
  • System component design
  • Service boundary definition
  • API and interface contracts
  • Technology stack selection
04
🔄
Refinement
Implement using TDD Red-Green-Refactor cycles. Debug, secure, and optimize through iterative improvement.
  • TDD Red-Green-Refactor cycles
  • Security review and hardening
  • Performance optimization
  • Code quality and refactoring
05
✅
Completion
Integrate, document, and validate against requirements. Deploy with monitoring and continuous improvement loops.
  • Integration and end-to-end testing
  • Documentation generation
  • Deployment and monitoring
  • Validation against acceptance criteria

CORE PRINCIPLES

Built on first principles.

✓
Modular Design
Keep files under 500 lines. Clear separation of concerns. Bounded contexts that scale independently.
✓
Test-Driven Development
Always write tests before implementation. Red-Green-Refactor is non-negotiable in the Refinement phase.
✓
Environment Safety
Never hardcode secrets or environment variables. Credentials stay out of code, always.
✓
Parallel Execution
Concurrent agent coordination delivers 2.8–4.4× speed improvements over sequential workflows.
✓
Memory Integration
Persistent knowledge sharing across agents and sessions via AgentDB vector memory.
✓
Quality First
Every task ends with attempt_completion. Comprehensive reviews and validation are mandatory.

QUICK START

Get running in seconds.

# Run SPARC orchestrator
npx claude-flow sparc "build complete authentication system"

# Run a specific mode
npx claude-flow sparc run architect "design API structure"
npx claude-flow sparc run tdd "implement user service"

# Full TDD workflow
npx claude-flow sparc tdd "implement user authentication"

# List all modes
npx claude-flow sparc modes --verbose
// Run SPARC orchestrator via MCP (preferred in Claude Code)
mcp__claude-flow__sparc_mode {
  mode: "sparc",
  task_description: "build complete authentication system"
}

// Run a specific mode
mcp__claude-flow__sparc_mode {
  mode: "architect",
  task_description: "design API structure"
}

// TDD workflow
mcp__claude-flow__sparc_mode {
  mode: "tdd",
  task_description: "implement user authentication",
  options: { workflow: "full" }
}
# Global install
npm install -g ruflo

# Or: add as Claude Code MCP server
claude mcp add ruflo -- npx -y ruflo@latest mcp start

# Then use locally
ruflo sparc "build complete auth system"
ruflo agent spawn -t architect
ruflo swarm init
